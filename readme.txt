A Plugin for Wordpress that displays additional information to help the developer.

== Description ==
<p>
DebugWp will place a new toolbar at the bottom of your wordpress site only visible to Administrators that shows information about any particular page you are looking at. 

The Wordpress Debug information includes :

<ul>
  <li>Total Queries</li>
  <li>Execution Time</li>
  <li>Object Information</li>
  <li>Page Type</li>
  <li>Front Page</li>
  <li>Post Type</li>
  <li>Page Template</li>
  <li>Queued Scripts</li>
  <li>Done Scripts</li>
  <li>Loaded Scripts</li>
  <li>Loaded Styles</li>
  <li>Site Information</li>
  <li>Bot Status</li>
  <li>Posts Per Page</li>
  <li>Active Theme<br />
  </li>
</ul>
You can read more about <a href="http://www.soapmedia.co.uk/debugwp-wordpress-debugger-for-developers">DebugWP on our blog article</a>
</p>
== Installation ==

1. Upload the `debugwp` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

The debug toolbar will appear at the bottom of front end pages if you are logged in as an administrator.
